# V2 Roadmap - Tiny Fixers

## High Priority

### New Puzzle Types
- [ ] **Match-3 Variant** - Swap tiles to match colors/patterns
- [ ] **Slide Puzzle** - Move pieces along tracks
- [ ] **Stack Puzzle** - Balance and stack items
- [ ] **Connect Puzzle** - Draw lines between matching items

### Content Expansion
- [ ] 30 additional levels (10 per existing type)
- [ ] New hub locations (Cafe, Bookstore, Park)
- [ ] 10 new characters with unique personalities
- [ ] Seasonal/holiday-themed level packs

### Monetization
- [ ] Rewarded video ad placement
  - Double coins after level completion
  - Free continue on failure
  - Bonus daily reward
- [ ] Cosmetic IAPs
  - Character skins/outfits
  - UI themes
  - Particle effects
- [ ] Battle pass-style reward track
